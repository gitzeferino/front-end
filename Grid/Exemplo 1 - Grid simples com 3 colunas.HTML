Grid com 3 colunas

Explicação:
A div.grid-container é o container do grid.
A propriedade display: grid define que vamos usar grid layout.
grid-template-columns: 1fr 1fr 1fr define 3 colunas com largura igual (1 parte para cada uma).
gap: 10px define o espaçamento entre os elementos da grade.
Cada div.grid-item representa um item dentro do grid e será distribuído nas colunas automaticamente.
Ideal para: criar layouts de cartões, galerias, ou distribuições regulares.
